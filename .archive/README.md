## Archive Retention Policy

**Keep Archived**:
- Milestone completion reports (historical record)
- ADR supersession history (if ADRs are updated)
- Major architectural decisions documentation
- Metrics and benchmarks from completed work

**Permanently Delete** (not currently applicable):
- Duplicate files
- Incomplete/abandoned work with no historical value
- Temporary scratch files
